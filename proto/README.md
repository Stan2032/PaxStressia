# proto/ — Phase 0 web prototype (lands v0.3)

A single-file HTML/JS playable of *Sahel-lite* — 6 nodes, 3 systems only
(legitimacy triangle, network growth, ~9 initiatives), persistent saves,
runs in a phone browser. It consumes the same `rules/*.json` as the sim.

The question it answers: **is the core loop fun in 20 minutes?**
(DESIGN.md §13.3, Phase 0.)
