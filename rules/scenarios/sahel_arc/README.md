# Scenario 1 — "The Arc" (Sahel, 2012–2026) — lands v0.4

Full ~12-node map (adds Ménaka, Niamey, northern Burkina, Tahoua, Lake Chad
rim, coastal-spillover marker), the Tuareg national front as a third faction,
the mercenary-patron actor, and the historical beat deck (2012 northern
collapse → 2013 intervention window → coup cascade → ECOWAS rupture → AES →
2026 insurgent-alliance offensives). See DESIGN.md §12.

The top-level `rules/*.json` currently hold the **Sahel-lite** development
dataset (6 nodes) used by the sim skeleton, tests, and the v0.3 prototype.
